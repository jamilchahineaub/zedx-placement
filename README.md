# ZED-X Dual Camera Placement Experiment
Isaac Sim 5.1 + ZED SDK 5.3.1

## Structure
- isaac/   — scripts that run under Isaac's Python
- zed/     — scripts that run under system Python + pyzed
- analysis/— plain Python, no special runtime
- config/  — machine-specific paths and experiment parameters
- results/ — per-layout CSVs and results.csv
